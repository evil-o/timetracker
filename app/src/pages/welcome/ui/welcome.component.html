Welcome to the time tracker app!

Your data is stored locally, in your browser. You must backup data yourself, use the cog icon in the navbar.

New features

Tags part 2

There's now a rudimentary aggregation of activities per tag in the week view.

Tags part 1

You can now add tags to activity descriptions. Just add them by prefixing them with #. These can be used in a variety of ways, for example, to associate acitvities with projects. For example, you can log an activity Meeting and use a description Regular sync up for project #Foo, where Foo is the name of the project.

Note that tags must be one word, i.e., cannot contain spaces.

One of the next steps for this feature is to provide summaries for the time spent per tag. Together, this will allow you to track times for projects or any other activities you may want to group outside of the regular acitivty types.
